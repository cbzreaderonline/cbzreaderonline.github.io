About the project

Comics deserve a reader that gets out of the way.

CBZ Reader Online was built around one simple idea: opening a comic archive shouldn't require installing anything. A CBZ file is just a ZIP archive of image pages, and a CBR file is the same idea using RAR compression, so a well-built webpage can unpack and display them exactly as well as a native desktop application, without asking you to trust a download or wait for a server to finish uploading and converting your file.

This reader runs entirely on the device you're using right now. When you drop a .cbz file into the box above, your browser reads it, extracts the image pages using client-side unzip logic, sorts them naturally, and hands them straight to the viewer. Nothing is transmitted anywhere. That matters if you're reading on a work laptop with restricted network access, on a shared or public computer, or if you simply don't want a comic collection uploaded to a stranger's server just to flip through a few pages.

We built it feature-first rather than app-first. That means the reading experience, page navigation, thumbnails, fit modes, keyboard shortcuts, and dark mode, was designed before anything else, and everything unnecessary was left out. There's no account system, no ads blocking the page, and no bloated framework slowing down load times. The result is a page that opens fast on a five-year-old laptop and just as fast on a mid-range Android phone.

CBZ Reader Online is meant to answer the same question people search for constantly: "what is the best cbz reader?" Our answer is that the best reader is the one that opens instantly, works on whatever device you already own, Windows, Mac, Linux, Chrome, or Android, and never gets in the way of the actual reading. Whether you're catching up on a webcomic archive, revisiting an old manga scanlation, or organizing a personal digital comic library, this tool is meant to be the fastest path from file to first page.

A CBZ file is a ZIP archive containing a sequence of image files, usually JPG or PNG, named so they sort in reading order. It's a packaging format used for digital comics and manga. Any tool that can open a ZIP file can technically open a CBZ, but a proper reader adds page navigation, zoom, and view modes on top.

Both formats store the same thing, a folder of comic page images, but they use different compression. CBZ wraps the pages in a ZIP archive, while CBR wraps them in a RAR archive. RAR compression can be slightly smaller for some files, but ZIP is an open, license-free format, which is why CBZ is more common for freely shared and self-published comics.

Read without internet

Use it offline too.

This site can be used completely without an internet connection once you have visited it a single time. Here is how that works and how to set it up.

  1. Open this website once while you are connected to the internet. This lets your browser save the app files needed to run it later.
  2. On desktop, you can also click the install icon in your browser's address bar, or use the browser menu and choose an option like Install App or Add to Home Screen. On Android in Chrome, use the menu and choose Add to Home Screen. On iPhone in Safari, use the Share button and choose Add to Home Screen.
  3. From then on, you can open the app icon or revisit this page even with no internet connection, no Wi-Fi, and no mobile data, and the reader will still load normally.
  4. Comic files always open from your local storage or a connected drive, so reading a file offline works exactly the same as reading it online, since your files were never uploaded anywhere in the first place.

A quick note: the very first load of any page still needs an internet connection so your browser can download and cache it. After that first visit, this page and its reader keep working offline until you clear your browser's site data.
